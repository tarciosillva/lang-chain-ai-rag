1.Baixar o repo

2. Criar o .env

3.No diretório do repositório, crie o ambiente virtual usando o comando: python3 -m venv venv

4. Criar arquivo de configuração de acesso do gcp

5. source venv/bin/activate

6. sudo apt update
   sudo apt install ffmpeg

7. pip install -r requirements.txt

8. python create_database.py

9. Sair do venv: deactivate
 
10. docker compose up -d
