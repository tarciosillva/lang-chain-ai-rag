1.Baixar o repo

2. Criar o .env

3.No diretório do repositório, crie o ambiente virtual usando o comando: python3 -m venv venv

4. source venv/bin/activate

5. sudo apt update
   sudo apt install ffmpeg

6. pip install -r requirements.txt

7. python create_database.py

8. Sair do venv: deactivate
 
9. docker compose up -d
